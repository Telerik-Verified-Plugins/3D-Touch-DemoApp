3D Touch Plugin Demo

The 3D Touch plugin allows you to add static and/or dynamic actions to the homescreen icon. You'll also be able to enable 'link preview' for external links in your app.

Custom plugins don't work in the simulator and the AppBuilder companion app, so inside Telerik AppBuilder choose Run - Build - iOS/Android - App package.

Check availability

Only iPhones 6s/6s+ and up are supported.

Adding dynamic home icon actions

Check the documentation in case you need static actions.

Link preview (disabled by default)

By pressing an external link your app will do what it always did, but by pressing a bit harder you'll see a preview pop up. Pressing even harder will launch the link in Safari.
